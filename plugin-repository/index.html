🚀 WoL Waker Plugin Repository

A Jellyfin plugin that automatically wakes up archival storage servers

📦 Installation

Add this repository to your Jellyfin instance:

https://remiverhoeven.github.io/jellyfin_wol_plugin/

Steps:

  1. Go to Jellyfin Dashboard → Plugins → Repositories
  2. Click "Add Repository"
  3. Enter the URL above
  4. Go to Plugins → Available
  5. Find "WoL Waker" and click Install

🔌 Plugin Features

  • Wake-on-LAN: Automatically wake archival storage servers
  • Power Monitoring: Integration with power monitoring APIs
  • Smart Wake Logic: Prevents unnecessary wake attempts
  • User Notifications: Keep users informed of server status

📋 Requirements

  • Jellyfin 10.9.0 or higher
  • .NET 8.0 runtime
  • Wake-on-LAN capable network card

🔄 Auto-Updates

This plugin repository automatically updates when new versions are pushed. Users just need to restart Jellyfin to get the latest version.

📁 Repository Files

  • manifest.json - Plugin repository manifest
  • plugin-packages/ - Plugin ZIP packages
